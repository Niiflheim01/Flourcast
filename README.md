# Flourcast: An Automated Production and Demand Forecasting System

A mobile-first React Native application designed for small, local bakeries to track sales, manage inventory, and predict daily demand to minimize waste and stockouts.

### Prerequisites
- Node.js (v16 or higher)
- Expo CLI: `npm install -g @expo/cli`
- Expo Go app on your phone

### Installation
```bash
# Clone the repository
git clone https://github.com/Niiflheim01/flourcast.git
cd flourcast

# Install dependencies
npm install

# Start development server
npm start
```

### Preview with Expo Go
1. Install **Expo Go** app on your phone
2. Scan the QR code that appears in your terminal
3. The app will load directly on your device!
